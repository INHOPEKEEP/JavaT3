## 实验目的

模拟学生选课系统，输出学生选课，退课信息。

## 实验要求

* 有人员，学生，教师，课程及测试类
* 人员有编号，性别，姓名等属性，
* 学生，教师继承人员类，另外还拥有自己特有的属性
* 测试类，输出学生选课，退课具体信息
